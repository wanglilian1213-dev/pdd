# UltraQA Matrix: Writing Extreme Cases 2026-05-28

## Goal

Continue the broad adversarial pass for the pindaidai writing system after five critic agents reviewed current gaps. This pass focuses on cases where the site could look successful while the final academic output is unsafe, wrong, privacy-leaking, or based on hidden spreadsheet data.

## Safety Bounds

- Do not print secrets or inspect browser cookies/password stores.
- Prefer local automated tests for broad coverage.
- Treat production upload-to-download flow as unproven unless a real task is created, completed, downloaded, opened, inspected, and cleaned up.

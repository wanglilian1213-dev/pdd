# Chart Examples

This folder contains two small scripts that generate four SVG charts each:

- scatter chart
- line chart
- bar chart
- pie chart

Run the Python version:

```bash
python3 chart_examples/python_charts.py
```

Run the TypeScript version:

```bash
./拼代代前端文件/node_modules/.bin/tsx chart_examples/typescript_charts.ts
```

Generated files are saved under:

- `chart_examples/output/python/`
- `chart_examples/output/typescript/`
